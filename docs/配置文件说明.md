## 配置文件结构

配置文件采用简单的 `key=value` 格式，支持注释。

```properties
# 这是注释
key=value
```

**规则：**
- 以 `#` 开头的行为注释
- 空行会被忽略
- 格式：`key=value`
- 不支持多行值

## 应用基本信息

### appName

- **类型**：字符串
- **必填**：是
- **说明**：应用内部名称，用于标识应用
- **示例**：`appName=我的WebView`

### appDisplayName

- **类型**：字符串
- **必填**：是
- **说明**：应用显示名称，在设备上显示的名称
- **限制**：
  - Android：无特殊限制
  - iOS：最多 30 字符
- **示例**：`appDisplayName=MyWebView`

### appId

- **类型**：字符串
- **必填**：是
- **说明**：应用唯一标识符，采用反向域名格式
- **格式**：`com.company.appname`
- **注意**：一旦发布到应用商店，不可更改
- **示例**：`appId=com.mywebviewapp`

### appVersion

- **类型**：字符串
- **必填**：是
- **说明**：应用版本号，建议使用语义化版本
- **格式**：`major.minor.patch`（如 1.0.0）
- **示例**：`appVersion=1.0.0`

### buildNumber

- **类型**：整数
- **必填**：否（**推荐不填，自动生成**）
- **说明**：构建号，每次发布必须递增
- **自动生成**：如果不指定，会自动使用当前时间生成（格式：`MMDDHHmmss`）
- **规则**：必须大于之前所有版本的 buildNumber
- **示例**：
  - 不指定：自动生成如 `1218143045`（12月18日14:30:45）
  - 手动指定：`buildNumber=1`（不推荐）
- **详细说明**：参考 [自动生成BuildNumber说明](./自动生成BuildNumber说明.md)

## 构建配置

### buildAndroid

- **类型**：布尔值
- **必填**：是
- **说明**：是否构建 Android 应用
- **可选值**：`true` / `false`
- **默认值**：`true`
- **示例**：`buildAndroid=true`

### buildIOS

- **类型**：布尔值
- **必填**：是
- **说明**：是否构建 iOS 应用
- **可选值**：`true` / `false`
- **默认值**：`true`
- **示例**：`buildIOS=true`

### isDebug

- **类型**：布尔值
- **必填**：是
- **说明**：构建模式
- **可选值**：
  - `true`：Debug 模式，用于开发和测试
  - `false`：Release 模式，用于生产发布
- **区别**：
  - Debug 模式不需要签名证书
  - Release 模式需要配置正式证书
- **示例**：`isDebug=true`

## WebView 配置

### loadUrl

- **类型**：字符串（URL）
- **必填**：是
- **说明**：WebView 加载的初始 URL
- **格式**：完整的 URL，包括协议（http:// 或 https://）
- **示例**：`loadUrl=https://www.baidu.com`

### enableJavaScript

- **类型**：布尔值
- **必填**：否
- **说明**：是否启用 JavaScript
- **默认值**：`true`
- **建议**：现代网页通常需要 JavaScript
- **示例**：`enableJavaScript=true`

### enableDOMStorage

- **类型**：布尔值
- **必填**：否
- **说明**：是否启用 DOM 存储（localStorage、sessionStorage）
- **默认值**：`true`
- **示例**：`enableDOMStorage=true`

### enableCache

- **类型**：布尔值
- **必填**：否
- **说明**：是否启用缓存
- **默认值**：`true`
- **效果**：启用后可加快页面加载速度
- **示例**：`enableCache=true`

### allowFileAccess

- **类型**：布尔值
- **必填**：否
- **说明**：是否允许访问本地文件
- **默认值**：`false`
- **安全提示**：出于安全考虑，建议保持为 false
- **示例**：`allowFileAccess=false`

### allowContentAccess

- **类型**：布尔值
- **必填**：否
- **说明**：是否允许访问内容 URI（Android）
- **默认值**：`false`
- **示例**：`allowContentAccess=false`

### allowFileAccessFromFileURLs

- **类型**：布尔值
- **必填**：否
- **说明**：是否允许 file:// URL 访问其他 file:// URL（Android）
- **默认值**：`false`
- **安全提示**：建议保持为 false
- **示例**：`allowFileAccessFromFileURLs=false`

### allowUniversalAccessFromFileURLs

- **类型**：布尔值
- **必填**：否
- **说明**：是否允许 file:// URL 访问所有源（Android）
- **默认值**：`false`
- **安全提示**：建议保持为 false
- **示例**：`allowUniversalAccessFromFileURLs=false`

### mixedContentMode

- **类型**：字符串
- **必填**：否
- **说明**：混合内容模式（HTTPS 页面加载 HTTP 资源）
- **可选值**：
  - `NEVER`：不允许
  - `ALWAYS_ALLOW`：总是允许
  - `COMPATIBILITY_MODE`：兼容模式
- **默认值**：`NEVER`
- **推荐**：使用 `NEVER` 以提高安全性
- **示例**：`mixedContentMode=NEVER`

### userAgentString

- **类型**：字符串
- **必填**：否
- **说明**：自定义 User Agent
- **默认值**：空（使用系统默认）
- **用途**：某些网站可能需要特定的 User Agent
- **示例**：`userAgentString=MyApp/1.0`

### loadingTextColor

- **说明**：启动页文字颜色
- **格式**：十六进制颜色代码
- **默认值**：`#FFFFFF`
- **示例**：`loadingTextColor=#FFFFFF`

### loadingText

- **类型**：字符串
- **必填**：否
- **说明**：启动页显示的文字
- **默认值**：`加载中...`
- **示例**：`loadingText=加载中...`

- **必填**：否
- **说明**：目标 Android SDK 版本

- **建议**：使用最新的稳定版本
- **示例**：`androidTargetSdkVersion=34`

### androidCompileSdkVersion

- **类型**：整数
- **必填**：否
- **说明**：编译 SDK 版本
- **默认值**：`34`
- **规则**：必须 >= targetSdkVersion
- **示例**：`androidCompileSdkVersion=34`

### androidApplicationId

- **类型**：字符串
- **必填**：否
- **说明**：Android 应用包名
- **默认值**：使用 `appId`
- **示例**：`androidApplicationId=com.mywebviewapp`

### androidVersionCode

- **类型**：整数
- **必填**：否
- **说明**：Android 版本代码
- **默认值**：使用 `buildNumber`
- **规则**：每次更新必须递增

### androidVersionName

- **类型**：字符串
- **必填**：否
- **说明**：Android 版本名称
- **默认值**：使用 `appVersion`

### androidKeyAlias

- **类型**：字符串

- **示例**：`iosDeploymentTarget=13.0`

### iosBundleId

- **类型**：字符串
- **必填**：否
- **说明**：iOS Bundle Identifier
- **默认值**：使用 `appId`
- **示例**：`iosBundleId=com.mywebviewapp`

### iosBundleDisplayName

- **类型**：字符串
- **必填**：否
- **说明**：iOS 显示名称
- **默认值**：使用 `appDisplayName`
- **限制**：最多 30 字符
- **示例**：`iosBundleDisplayName=MyWebView`

### iosBundleVersion

- **类型**：字符串
- **必填**：否
- **说明**：iOS Bundle 版本
- **默认值**：使用 `appVersion`
- **示例**：`iosBundleVersion=1.0.0`

### iosBuildNumber

- **类型**：字符串或整数
- **必填**：否
- **说明**：iOS Build Number
- **默认值**：使用 `buildNumber`
- **规则**：每次更新必须递增
- **示例**：`iosBuildNumber=1`

### iosTeamId

- **类型**：字符串
- **必填**：Release 模式必填
- **说明**：Apple Developer Team ID
- **格式**：10 位字符
- **获取方式**：Apple Developer 账号 > Membership
- **示例**：`iosTeamId=ABCDE12345`

### iosCertificateName

- **类型**：字符串
- **必填**：Release 模式必填
- **说明**：签名证书名称（占位符）
- **格式**：`Apple Distribution: Your Name (TEAM_ID)`
- **安全提示**：实际证书存储在 GitHub Secrets
- **示例**：`iosCertificateName=PLACEHOLDER_CERTIFICATE_NAME`

### iosProvisioningProfile

- **类型**：字符串
- **必填**：Release 模式必填
- **说明**：Provisioning Profile UUID（占位符）
- **安全提示**：实际 Profile 存储在 GitHub Secrets
- **示例**：`iosProvisioningProfile=PLACEHOLDER_PROVISIONING_PROFILE`

### iosExportMethod

- **类型**：字符串
- **必填**：否
- **说明**：IPA 导出方式
- **可选值**：
  - `app-store`：上架 App Store
  - `ad-hoc`：Ad Hoc 分发
  - `development`：开发测试
  - `enterprise`：企业分发
- **默认值**：`app-store`
- **示例**：`iosExportMethod=app-store`

### iosSku

- **类型**：字符串
- **必填**：否
- **说明**：App Store Connect 的 SKU（唯一标识符）
- **默认值**：如果不填，将使用 `appId` 替换 `.` 为 `-` 作为默认值
- **用途**：用于 App Store Connect API 创建应用时的唯一标识
- **示例**：`iosSku=com-xlab-myapp`

### iosPrimaryLocale

- **类型**：字符串
- **必填**：否
- **说明**：App Store Connect 的主要语言
- **默认值**：`zh-Hans`
- **常见值**：
  - `zh-Hans`：简体中文
  - `zh-Hant`：繁体中文
  - `en-US`：美国英语
  - `ja`：日语
  - `ko`：韩语
- **示例**：`iosPrimaryLocale=zh-Hans`

### iosLocales

- **类型**：字符串（逗号分隔）
- **必填**：否
- **说明**：支持的语言列表
- **默认值**：`zh-Hans,en-US`
- **用途**：为这些语言创建本地化信息
- **示例**：`iosLocales=zh-Hans,en-US,ja`

## 高级配置

### enableDebugging

- **类型**：布尔值
- **必填**：否
- **说明**：是否启用 WebView 调试
- **默认值**：`true`
- **效果**：
  - Android：可以使用 Chrome DevTools 调试
  - iOS 16.4+：可以使用 Safari Web Inspector 调试
- **建议**：开发时启用，生产时禁用
- **示例**：`enableDebugging=true`

### clearCacheOnStart

- **类型**：布尔值
- **必填**：否
- **说明**：启动时是否清除缓存
- **默认值**：`false`
- **用途**：确保每次加载最新内容
- **示例**：`clearCacheOnStart=false`

### enableZoom

- **类型**：布尔值
- **必填**：否
- **说明**：是否允许用户缩放页面（双指缩放）
- **默认值**：`true`
- **效果**：
  - `false`：完全禁用页面缩放，通过 JavaScript 注入 viewport meta 标签实现
  - `true`：允许用户通过双指手势缩放页面
- **实现**：
  - Android：设置 `setSupportZoom`、`useWideViewPort`、`loadWithOverviewMode` 并注入 viewport
  - iOS：通过 WKUserScript 注入 viewport meta 标签
- **建议**：如果你的网页内容已做好响应式设计，建议设置为 `false` 以获得更好的用户体验
- **示例**：`enableZoom=false`

### enableBuiltInZoomControls

- **类型**：布尔值
- **必填**：否
- **说明**：是否显示内置缩放控件（Android 专属）
- **默认值**：`false`
- **注意**：此配置只控制是否显示缩放控件按钮，不影响手势缩放
- **建议**：通常设置为 `false`，配合 `enableZoom=false` 完全禁用缩放
- **示例**：`enableBuiltInZoomControls=false`

### supportMultipleWindows

- **类型**：布尔值
- **必填**：否
- **说明**：是否支持多窗口
- **默认值**：`false`
- **用途**：允许网页打开新窗口
- **示例**：`supportMultipleWindows=false`

## App Store 元数据配置

以下配置用于 App Store Connect API 自动创建和更新应用元数据。

### appSubtitle

- **类型**：字符串
- **必填**：否
- **说明**：应用副标题（App Store 显示）
- **限制**：最多 30 个字符
- **示例**：`appSubtitle=轻松学习的小游戏`

### appDescription

- **类型**：字符串
- **必填**：否
- **说明**：应用描述（通用）
- **用途**：如果没有设置特定语言的描述，则使用此项
- **限制**：最多 4000 个字符
- **示例**：`appDescription=这是一个有趣的学习应用`

### appDescription_{语言代码}

- **类型**：字符串
- **必填**：否
- **说明**：特定语言的应用描述
- **格式**：`appDescription_` + 语言代码（将 `-` 替换为 `_`）
- **示例**：
  - `appDescription_zh_Hans=简体中文描述`
  - `appDescription_en_US=English description`

### appKeywords

- **类型**：字符串（逗号分隔）
- **必填**：否
- **说明**：应用关键词（通用）
- **限制**：最多 100 个字符
- **用途**：用于 App Store 搜索
- **示例**：`appKeywords=学习,游戏,教育`

### appKeywords_{语言代码}

## App Store 截图配置

以下配置用于自动生成和上传 App Store 截图。

### enableScreenshotUpload

- **类型**：布尔值
- **必填**：否
- **说明**：是否启用截图自动生成和上传
- **默认值**：`true`
- **效果**：
  - `true`：自动从 `splashScreen` 生成截图并上传到 App Store Connect
  - `false`：跳过截图生成和上传
- **示例**：`enableScreenshotUpload=true`

### screenshotDeviceTypes

- **类型**：字符串（逗号分隔）
- **必填**：否
- **说明**：要生成的设备类型列表
- **默认值**：`iPhone_6.7,iPad_12.9_3rd`
- **可选值**：
  - `iPhone_6.7`：iPhone 14/15 Pro Max（1290x2796）**必需**
  - `iPhone_6.5`：iPhone 11/XS Pro Max（1242x2688）
  - `iPhone_5.5`：iPhone 8 Plus（1242x2208）
  - `iPad_12.9_3rd`：iPad Pro 12.9"（2048x2732）
  - `iPad_12.9_2nd`：iPad Pro 12.9"（2048x2732）
- **建议**：至少包含 `iPhone_6.7`（Apple 必需）和 `iPad_12.9_3rd`（如果支持 iPad）
- **示例**：`screenshotDeviceTypes=iPhone_6.7,iPhone_6.5,iPad_12.9_3rd`

### screenshotAddText

- **类型**：布尔值
- **必填**：否
- **说明**：是否在截图上添加应用名称和副标题
- **默认值**：`false`
- **效果**：
  - `true`：在截图底部添加 `appDisplayName` 和 `appSubtitle`
  - `false`：只显示图片，不添加文字
- **用途**：可以让截图更有吸引力
- **示例**：`screenshotAddText=false`
